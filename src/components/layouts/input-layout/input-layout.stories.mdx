## InputLayout

The InputLayout is a versatile block container meant to create various form layouts. Because it is meant to wrap single elements use this together with the FormLayout component.

##### This container gives the ability to:

-   control the size of the columns
-   keep the spacing consistent

### Props

-   children?: React.ReactNode;
-   align?: 'center' | 'start' | 'end';
    -   Align content vertically inside the InputLayout
-   isExpanded?: boolean;
    -   Use the isExpanded modifier on the element if you want to fill up the remaining space
-   colSize?: 1 | 2 | 3 | 4 | 5 | 6;
    -   Use the colSize modifier to give a width, otherwise it defaults to auto width: Note: smaller colsizes will not override a child thats has already a larger width.

```js
import { InputLayout } from '@ksyos/design-system';
```

export const Block = styled.div`
    display: block;
    border-radius: 2px;
    height: 40px;
    background: #ddd;
    display: flex;
    justify-content: center;
    align-items: center;
    padding: 1rem 2rem;
`;

export const StyledFormLayout = styled(FormLayout)`
    height: 80px;
`;
