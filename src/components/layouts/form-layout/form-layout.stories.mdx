## FormLayout

The FormLayout is a block container meant to create various form layouts. When component receieven item props, all children will be wrapped inside InputLayouts. This allows the spacing and sizing to be consistent. Use this if you want grouped equal columns If you need more flexibility you can use the InputLayout to control the layout of the components children.

##### This container gives the ability to:

-   keep the spacing consistent
-   stack multiple children as a list

### Props

-   children?: React.ReactNode;
    -   Use this variant if all children should be displayed in unequal columns, use InputLayout
-   items?: JSX.Element[];
    -   Use this variant if all children should have equal column sizes and spacing
-   justify?: 'center' | 'left' | 'right' | 'space-between';
    -   Controls the horizontal alignment of the children within the container.
-   layout?: 'grouped' | 'condensed' | 'stacked';
    -   Controls the layout of the children:
    -   Use grouped when the children should be displayed without any spacing in between
    -   Use condensed to decrease the padding between columns
    -   Use stacked to display the children in a vertical column
-   colSize?: 1 | 2 | 3 | 4 | 5 | 6
    -   Use the colSize modifier to give a width, otherwise it defaults to auto width.

```js
import { FormLayout } from '@ksyos/design-system';
```

export const Block = styled.div`
    display: block;
    border-radius: 2px;
    height: 40px;
    background: #ddd;
    display: flex;
    justify-content: center;
    align-items: center;
    padding: 1rem 2rem;
    margin-bottom: 1rem;
`;
