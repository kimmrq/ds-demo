## Questionnaire Form

To build a form layout specific for the triage questionnaire: A set of 3 components are needed:

-   QuestionnaireForm - which handles the submit
-   QuestionnaireIntro - Presentational component for the large text
-   QuestionnaireFormSection - Presentational component to divide the form into multiple sections

```js
import {
    QuestionnaireForm,
    QuestionnaireIntro,
    QuestionnaireFormSection,
} from '@ksyos/design-system';
```

### Form elements

The rest of the form layout can made out of multiple base atoms and molecules from the Design system.

-   FormSection
-   FormGroup
-   TextInput
-   ButtonRow
-   Button
