# InputCheckmark

```js
import { InputCheckmark } from '@ksyos/design-system';
```

InputCheckmark displays either a single or multiselect button

-   InputCheckmark extends on RadioProps
-   InputCheckmark defaults to type checkbox
-   When used as a single select variant, InputCheckmark can be used inside RadioGroup. See story examples inside a RadioGroup
-   Display InputCheckmarks either horizontally or vertically. For examples see stories RadioGroup and InputCheckmark
-   Additional "isFullWidth" prop can be used to display a fullwidth variant. Use, for example, on smaller viewports in a vertical layout
