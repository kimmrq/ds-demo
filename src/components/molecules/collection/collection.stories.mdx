# Collection

### Default example of a Collection

Collections allow you to group list objects together.

-   Use CollectionItem to display list items
-   Use CollectionLink to display links. When used as React Router Link use `as={Link}` prop.
-   Both components can accept a mediaItem

When used in a card, set the spacing of the card to "none"
