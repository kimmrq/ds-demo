# ButtonRow

Consistent spacing between buttons.

```js
import { ButtonRow } from '@ksyos/design-system';
```

### Stacked at Small

Best used with "full-below-small" on the children.

Note: this reverses the order of the buttons on small screens as well.

    <Story name="Stacked at Small">
